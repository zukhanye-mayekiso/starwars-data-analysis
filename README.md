# starwars-data-analysis
Star Wars data analysis using R
This project explores the starwars dataset to analyze character attributes such as height , mass and gender. My goal and intention were to uncover patterns and relationships within the data using R. Overall ,this project demonstrates foundational data analysis skills including data cleaning, transformation, visualization, and interpretation using R.

Tools and Technologies used: R , tidyverse(dplyr , ggplot2)
Data prepation :
- loaded and explored the dataset
- handled missing values using drop_na()
- selected relevant variables for analysis
  
📊 Data Visualization
made a histogram to show the distribution of heights

created a boxplot to compare heights between genders.

created a scatter plot to investigate the connection between mass and height
🔍 Key Insights
The majority of characters are in a particular height range.

There is a positive correlation between height and mass.

There are differences in height amongst the various gender groups.
